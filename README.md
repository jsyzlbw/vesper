Vesper is a private, native iPhone assistant. Describe what you want in natural language; Vesper turns it into explicit, editable actions such as reminders, notifications, real alarms, calendar events, diary entries, tasks, and debug logs.

Vesper 是一个原生 iPhone 私人助理。你可以用自然语言描述计划，Vesper 会把意图转换成可检查、可编辑、可确认的提醒、通知、真闹钟、日历事件、日记、任务和调试日志。
